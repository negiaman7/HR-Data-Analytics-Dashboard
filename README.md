# HR-Data-Analytics-Dashboard
1. Project Title- HR-Data-Analytics-Dashboard 
A dynamic, interactive data visualization tool built to explore HR analytics (employee data, attrition, demographics).

2. Short Description-
   The Analytics Dashboard is a visually engaging, analytical Power BI report designed for HR Analytics Dashboard to analyze workforce metrics such as attrition rate, employee distribution, and salary trends. The dashboard helps identify high-attrition areas, evaluate diversity, and support strategic HR decision-making.

3. Tech Stack
The dashboard was built using the following tools and technologies.
   Power BI Desktop- Main tool to build the dashboard. Used for data modeling, DAX calculations, and visualizations
   Power Query Editor- Used for data cleaning and transformation. Handling missing values, changing data types, and creating calculated columns
   DAX (Data Analysis Expressions)- For writing measures such as: Attrition Rate, Total Employees, Average Salary, Department-wise KPIs
   Excel / CSV (Source Data)- Raw HR dataset is usually stored in Excel or CSV format. Imported into Power BI for analysis.

5. Project Purpose
   Track workforce health and trends
   Identify departments with high attrition
   Analyze salary and experience patterns
   Evaluate diversity metrics
   Support HR teams in strategic decision-making

6. Summary
   This project showcases an HR Analytics Dashboard built using Power BI to analyze key workforce metrics and support data-driven HR decision-making. The dashboard provides insights into attrition trends, employee demographics, salary patterns, and departmental performance. It includes KPI indicators, interactive visuals, and filters that allow users to explore HR data from multiple perspectives.

   The project involves data cleaning and transformation in Power Query, KPI creation using DAX measures, and the design of an interactive, business-ready dashboard. By converting raw HR data into actionable insights, the dashboard helps HR teams identify high-turnover areas, evaluate diversity, analyze compensation structures, and understand the overall health of the organization.
